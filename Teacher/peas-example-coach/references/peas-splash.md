# PEAS 開場品牌畫面（凡思陪練）

供 `peas-example-coach` 在**新工作階段第一則**訊息頂端使用。代理應**讀取本檔**後，依下列**順序**輸出（同一則訊息內、由上而下）：

1. **文字字標（必出）**：先輸出一行純文字品牌列（**不要**包在程式碼區塊內）。
2. **空一行**。
3. **對話用版面**：將下方「對話用版面」整段以**單一** fenced code block（語言標記建議 `text`）輸出，以保持等寬對齊。

輸出完上述畫面後，**同一則訊息內**須依 skill 主檔「**開場準備確認**」以**活潑、興奮**的口語邀請使用者開跑（非平淡公事問句）；**勿**在本畫面區塊內嵌入進度條或第一題。

## 文字字標（必出）

- **輸出格式**：在對話中輸出**一行**純文字，**不要**用程式碼區塊包起來。建議字面：

  `PEAS · 凡思陪練`

- **說明**：與下方 ASCII 框內 **PEAS**／標題列並存無妨；可依上下文微調（例如加上簡短歡迎），**不要**留空或略過品牌。

## 可替換欄位

| 佔位意義 | 建議 |
|----------|------|
| 標題列（chevron 右側第一行） | 例如 `PEAS Example Coach · 凡思陪練`；可含簡短版號字樣 |
| Session 列 | 例如 `Session: 本輪陪練剛開始` 或專案／使用者狀態一句話 |
| Mode 列 | 例如 `Mode: 思考格逐題引導`；可附選用指令提示（如 `/help`） |
| 歡迎語 | 可併入標題列或 Session 列，不必單獨置中一行 |
| 路徑列 | 專案根目錄路徑；超過左欄寬度時從左截斷保留開頭，尾端加 `…` |

## 對話用版面（複製此結構）

版面寬度約 78 字元；橘色邊框在純文字環境以 `┌┐└┘│─` 近似（與附檔 Claude Code splash 同構：左主區 + 直線 + 右側欄）。**Logo 區**改為 **線條構成的右向 chevron**（類 Gemini CLI 啟動畫面之 `>` 意象，但**只用** `/`、`\`、`|` 與空白，**不用**方塊像素 `█`）；左欄 chevron 右側三行對應「標題／Session／Mode」，可代換為當下專案路徑、歡迎語等（見「可替換欄位」）。

```
┌─────────────────────────────────────────────────────────────┬──────────────────────────┐
│                                                             │                          │
│      /|     PEAS Example Coach · 凡思陪練                       │Tips for getting started  │
│     / |     Session: 本輪陪練剛開始                                │一次只問一題，先說依據               │
│    /  |     Mode: 思考格逐題引導                                   │再一起補齊觀念。                  │
│   /   |                                                     │                          │
│    \  |                                                     │Recent activity           │
│     \ |                                                     │本輪陪練剛開始                   │
│      \|                                                     │                          │
│                                                             │                          │
│  ~/your/project/root/path…                                  │                          │
│                                                             │                          │
└─────────────────────────────────────────────────────────────┴──────────────────────────┘
```

### 線條 Logo（給代理微調時對齊附檔 Gemini CLI 風）

- **結構**：約 **7** 行；上半由 `/` 與 `|` 組成外張邊，下半由 `\` 與 `|` 收斂，整體為**線條**右向箭號，**禁止**改回方塊像素身軀。
- **對齊**：chevron 本體建議固定佔左欄前 **12** 字元寬區塊（`padEnd` 對齊），右側再接標題列；維持左欄總寬 **61**、右欄 **26**，與框線 `│` 垂直對齊。
- **文案列**：三行可依情境改為例如 `歡迎回來！`、`Signed in…`／`專案：…`、`Plan…`／`Mode…`；**路徑列**維持框內左下獨立一行（過長從左截斷＋結尾 `…`）。

## 終端機用（選用，ANSI）

若使用者在**終端機**執行腳本而非在 Cursor 對話中看畫面，可對**線條 chevron**加上**直向漸層**（上青藍 → 中紫 → 下洋紅／粉，近似 Gemini CLI logo），標題列用亮白、次要資訊用灰。**Cursor 聊天室不一定渲染 ANSI**，預設仍以「對話用版面」為主。

下列 Python 3 範例會印出 chevron（僅線條）＋右側三行白字＋重設；框線可再自行加上 `\033[38;5;208m` 等橘色。

```python
# 僅供參考：終端機啟動時列印（需支援 truecolor 的終端機）
RESET = "\033[0m"
BOLD = "\033[1m"
DIM = "\033[2m"

def rgb_fg(r: int, g: int, b: int) -> str:
    return f"\033[38;2;{r};{g};{b}m"

# 七行 chevron（與對話用版面同款線條）
lines = [
    "      /|",
    "     / |",
    "    /  |",
    "   /   |",
    "    \\  |",
    "     \\ |",
    "      \\|",
]
labels = [
    f"{BOLD}PEAS Example Coach · 凡思陪練{RESET}",
    f"{DIM}Session:{RESET} 本輪陪練剛開始",
    f"{DIM}Mode:{RESET} 思考格逐題引導",
]
# 漸層：青藍 (0,220,255) → 紫 (160,90,255) → 洋紅 (255,80,180)
for i, art in enumerate(lines):
    t = i / max(len(lines) - 1, 1)
    if t < 0.5:
        u = t * 2
        r = int(0 + (160 - 0) * u)
        g = int(220 + (90 - 220) * u)
        b = int(255 + (255 - 255) * u)
    else:
        u = (t - 0.5) * 2
        r = int(160 + (255 - 160) * u)
        g = int(90 + (80 - 90) * u)
        b = int(255 + (180 - 255) * u)
    suffix = ""
    if i < len(labels):
        suffix = "  " + labels[i]
    print(rgb_fg(r, g, b) + art + RESET + suffix)
```

代理**不必**在一般陪練對話中強制輸出 ANSI，除非使用者明確要求終端機版。
